How to use

For performance reasons, all icons require a base class and individual icon class. To use, place the following code just about anywhere. Be sure to leave a space between the icon and text for proper padding.

Always use only dedicated empty <span>s

Icon classes should only be used on dedicated <span>s that contain no text content or child elements. The <span> should also not have any other classes applied to it.
